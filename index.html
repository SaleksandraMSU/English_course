Course Overview

This course provides a comprehensive introduction to hydrological terrain analysis using SAGA GIS. Students will learn to process digital elevation models (DEMs), model water flow, delineate watersheds, compute hydrological indices, and create professional maps for environmental and hydrological applications.

Course Objectives

  • Learn how to perform hydrological terrain analysis using SAGA GIS
  • Understand DEM preprocessing and conditioning
  • Model flow direction, accumulation, and watershed basins
  • Compute hydrological indices (TWI, SPI, LS Factor)
  • Produce professional hydrological maps

Session 1: Introduction to DEM and SAGA GIS

Learning Objectives

  • Define what a Digital Elevation Model (DEM) is and why it’s essential for hydrology
  • Identify and distinguish between common DEM formats (GeoTIFF, ASCII, SAGA Grid)
  • Navigate the SAGA GIS interface and manage raster/vector data
  • Apply core morphometric tools: slope, aspect, curvature, and hillshading
  • Verify and assign correct coordinate reference systems (CRS)
  • Export analytical outputs for downstream use

Digital Elevation Models: Concepts and Characteristics

A Digital Elevation Model (DEM) is a raster-based representation of the Earth's surface, where each cell contains a numerical elevation value.

Common DEM Formats:

  • GeoTIFF (.tif)
  • ASCII Grid (.asc)
  • SAGA Grid (.sgrd / .sdat)

Introduction to the SAGA GIS Environment

SAGA GIS is a specialized system for terrain and hydrological analysis.

Data Manager Structure:

  • Grids – raster layers (elevation, slope, flow accumulation, etc.)
  • Shapes – vector layers (points, lines, polygons)
  • Tables, point clouds, and other data types

Practical Applications in Session 1

  • Importing DEM Data: File → Import → Import Raster; Modules → Import/Export Grids → Import Grid (GDAL)
  • Morphometric Analysis: Slope, Aspect, Curvature
  • Hillshading and Terrain Visualization: Terrain Analysis → Lighting, Visibility → Analytical Hillshading

Assignment

  1. Select a 20×20 km study area and obtain an appropriate DEM
  2. Import the DEM into SAGA GIS and verify its coordinate reference system
  3. Calculate terrain derivatives: Slope, Aspect
  4. Export the results as GeoTIFF for further use

Session 2: DEM Preprocessing & Hydrological Correction

Learning Objectives

  • Understand DEM Quality Issues
  • Apply Hydrological Conditioning Methods
  • Perform DEM Preprocessing in SAGA GIS
  • Prepare a Corrected DEM for Further Modeling

DEM Artifacts and Their Impact

  • Sinks (Depressions): Local closed basins where water becomes "trapped," preventing downstream flow
  • Flats: Areas with uniform elevation lacking a clear flow direction
  • Voids or Missing Data: Gaps in DEM coverage
  • Artificial Noise: Striping, spikes, or interpolation errors from remote-sensing datasets

Practical Procedures

  1. Identifying Sinks and Flats: Terrain Analysis → Preprocessing → Sinks Detection/Flat Detection
  2. Filling Sinks: Terrain Analysis → Preprocessing → Fill Sinks (Wang & Liu)
  3. Validating the Correction: Use Grid Calculator: DEM_filled - DEM_original
  4. Recomputing Morphometric Parameters: Recalculate Slope, Aspect, Curvature after correction

Output

Conditioned DEM ready for hydrological modeling.

Session 3: Flow Direction & Accumulation

Objective

Understand the concepts of hydrological modeling in a raster environment and perform key drainage network extraction processes using SAGA GIS.

The Hydrological Modeling Workflow

DEM → Flow Direction → Flow Accumulation → Stream Network

Flow Algorithms

D8 (Deterministic 8-node)

Water flows to the one steepest downhill neighbor. Simple and fast but can produce artificial parallel flow lines.

MFD (Multiple Flow Direction - FD8)

Water is distributed to multiple downhill neighbors proportionally to slope. More realistic for divergent flow.

Rho8 (Random 8-node)

Stochastic version of D8 with random choice between equal-slope cells. Produces more natural-looking networks.

Practical Steps in SAGA GIS

  1. Data Preparation: Load DEM and fill sinks
  2. Flow Direction: Terrain Analysis → Channels → Flow Direction (Parallel)
  3. Flow Accumulation: Terrain Analysis → Channels → Flow Accumulation (Top-Down)
  4. Channel Network Extraction: Terrain Analysis → Channels → Channel Network (set initiation threshold)
  5. Flow Path Length: Terrain Analysis → Channels → Flow Path Length

Session 4: Watershed Delineation

Concepts

  • Watershed: Area of land where all precipitation drains to a common outlet
  • Pour Point: The lowest point where water exits a watershed (outlet)
  • Sub-basins: Smaller drainage units within a larger watershed

Key Modules

  • Watershed Basins: Automated watershed delineation from DEM
  • Catchment Area (Parallel): Computes flow accumulation and drainage patterns
  • Shapes → Create Points Layer: Manual pour point creation

Workflow

  1. Automated Basin Delineation: Use Watershed Basins tool with filled DEM and flow direction
  2. Manual Delineation with Pour Points: Digitize pour points and extract upstream catchments
  3. Sub-basin Hierarchy: Use multiple pour points to create nested sub-basins

Output

Watershed polygons, sub-basin hierarchy (vector layers).

Session 5: Hydrological Indices

Key Hydrological Indices

Topographic Wetness Index (TWI)

Formula: TWI = ln(A / tanβ)
Meaning: Predicts soil moisture and saturation zones. High TWI = wetter areas (valleys, hollows).

Stream Power Index (SPI)

Formula: SPI = A × tanβ
Meaning: Estimates erosive power of overland flow. High SPI = high erosion potential.

LS Factor (RUSLE)

Meaning: Slope length and steepness factor for soil erosion modeling. Used in agricultural and conservation planning.

Calculation in SAGA GIS

  • TWI: Terrain Analysis → Hydrology → Topographic Wetness Index
  • SPI: Terrain Analysis → Hydrology → Stream Power Index
  • LS Factor: Terrain Analysis → Soil Erosion → LS Factor (RUSLE)

Output

Hydrological sensitivity maps (TWI, SPI, LS).
